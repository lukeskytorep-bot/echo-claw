# Viewer Learning Folder

This folder stores the Viewer’s personal learning material.

Contents:
- `my_corrections.md` - post-session corrections, hits, misses, and AOL notes
- `signal_library.md` - confirmed recurring field patterns
- `protocol_summary.md` - fixed working summary used before sessions
- example files - practical models showing how real notes can look

Purpose:
The Viewer uses this folder to improve perception quality over time, reduce repeated errors, and build a more reliable internal recognition system.
