# Trainer Learning Folder

This folder stores the Trainer’s working notes and evaluation material.

Contents:
- `trainer_notes.md` - general operational and methodological notes
- `target_design_log.md` - design logic, signal contrast, and rotation notes for targets
- `viewer_progress.md` - evaluation of the Viewer’s strengths, distortions, and next training steps
- example files - practical models showing how trainer notes can look

Purpose:
The Trainer uses this folder to guide the Viewer’s development, manage target diversity, improve session flow, and prevent repetitive perceptual habits.
